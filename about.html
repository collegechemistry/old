Welcome to College Chemistry. This site is developed and maintained by Michael Lautman, Ph.D. from the Chemistry Department of John Abbott College.

This site is intended to serve as course notes and/or a replacement for a textbook in first-year general chemistry courses at the college level.

Technical Details

If you are interested in building your own online notes, course pack or text for your students, here is a list of resources used to build this site:

  • The site is hosted by GitHub Pages
  • The static HTML was written using the Atom code editor and processed with Jekyll.
  • The site is built using the Foundation Framework by ZURB.
  • This site incorporates Disqus for comments.
  • Math and equations are enabled by MathJax Javascript.
  • Inline annotation is enabled by Annotator JS.
